## Overview

The `ToolsPanel` component provides a comprehensive interface for configuring image processing operations. It combines multiple tools (optimization, format conversion, resizing, and cropping) in a unified panel with a clear processing order.

## Features

- Image optimization toggle
- Output format selection (JPG, PNG, WEBP, original)
- Advanced resize controls via ResizeTool
- Advanced crop controls via CropTool
- Clear indication of processing order
- Brand-aware theming
- Disabled state support

## Processing order

<Info>
  Tools are applied in a specific order: **1) Optimization → 2) Resizing → 3) Cropping → 4) Format conversion**
</Info>

## Usage

### Basic usage

```jsx
import ToolsPanel from '@/components/Tools/ToolsPanel';

function MyComponent() {
  const [settings, setSettings] = useState({
    optimize: true,
    format: 'original',
    resize: {
      enabled: false,
      width: 800,
      unit: 'px',
      maintainAspectRatio: true,
    },
    crop: {
      enabled: false,
      cropType: 'square',
      position: 'center',
      width: 1000,
      height: 1000,
    },
  });

  const toggleOptimize = () => {
    setSettings(prev => ({ ...prev, optimize: !prev.optimize }));
  };

  const changeFormat = (format) => {
    setSettings(prev => ({ ...prev, format }));
  };

  const updateResizeSettings = (updates) => {
    setSettings(prev => ({
      ...prev,
      resize: { ...prev.resize, ...updates },
    }));
  };

  const updateCropSettings = (updates) => {
    setSettings(prev => ({
      ...prev,
      crop: { ...prev.crop, ...updates },
    }));
  };

  return (
    <ToolsPanel
      processingSettings={settings}
      toggleOptimize={toggleOptimize}
      changeFormat={changeFormat}
      updateResizeSettings={updateResizeSettings}
      updateCropSettings={updateCropSettings}
    />
  );
}
```

## Child components

The ToolsPanel composes two specialized tool components:

### ResizeTool

Handles image resizing configuration with support for:

- Enable/disable toggle
- Pixel or percentage units
- Width-based resizing
- Aspect ratio maintenance (always enabled)

<Accordion title="ResizeTool props">
  <ParamField path="resizeSettings" type="object" required>
    Current resize configuration.
  </ParamField>
  
  <ParamField path="updateResizeSettings" type="function" required>
    Callback to update settings.
  </ParamField>
  
  <ParamField path="disabled" type="boolean">
    Disable all controls.
  </ParamField>
</Accordion>

### CropTool

Handles image cropping configuration with support for:

- Enable/disable toggle
- Preset aspect ratios (square, 16:9, 4:3, 9:16, 3:4)
- Nine position options for crop placement
- Automatic dimension calculation based on crop type

## Format options

The format selector supports four output options:

<Tabs>
  <Tab title="Original">
    Maintains the original file format. Useful when you want to preserve format while applying other transformations.
  </Tab>
  
  <Tab title="JPG">
    Converts to JPEG format. Best for photographs and images with many colors. Smaller file sizes with lossy compression.
  </Tab>
  
  <Tab title="PNG">
    Converts to PNG format. Best for images with transparency or when lossless compression is needed.
  </Tab>
  
  <Tab title="WEBP">
    Converts to WebP format. Modern format with superior compression. Best for web delivery.
  </Tab>
</Tabs>

## Resize settings

### Units

- **Pixels (px):** Absolute size in pixels (max 5000px)
- **Percentage (%):** Relative to original size (1-100%)

### Behavior

<Note>
  Aspect ratio is always maintained. The width setting is used as the base measurement, and height is calculated proportionally.
</Note>

## Crop presets

The CropTool provides five preset aspect ratios:

| Preset | Aspect Ratio | Default Dimensions | Use Case |
|--------|--------------|-------------------|----------|
| Square | 1:1 | 1000x1000 | Profile pictures, Instagram posts |
| Horizontal (16:9) | 16:9 | 1600x900 | YouTube thumbnails, wide displays |
| Horizontal (4:3) | 4:3 | 1200x900 | Standard displays, presentations |
| Vertical (9:16) | 9:16 | 900x1600 | Instagram Stories, mobile portraits |
| Vertical (3:4) | 3:4 | 900x1200 | Portrait photography |

## Crop positions

Nine position options control where the crop is applied:

```
┌─────────┬─────────┬─────────┐
│ Top     │   Top   │   Top   │
│ Left    │ Center  │  Right  │
├─────────┼─────────┼─────────┤
│ Center  │ Center  │ Center  │
│ Left    │(Default)│  Right  │
├─────────┼─────────┼─────────┤
│ Bottom  │ Bottom  │ Bottom  │
│ Left    │ Center  │  Right  │
└─────────┴─────────┴─────────┘
```
